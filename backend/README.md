# Features

**Create a RESTful API with the following endpoints:**
- [ ] GET /api/products: Retrieve all products.
- [ ] /api/products/{id}: Retrieve a product by ID.
- [ ] POST /api/products: Add a new product.
- [ ] PUT /api/products/{id}: Update an existing product.
- [ ] DELETE /api/products/{id}: Delete a product.

**Define a Product model with the following fields:**
- [ ] 
```js
    Id (auto-generated)
    Name (string)
    Description (string)
    Price (decimal)
    StockQuantity (integer)
```

*Implement input validation for all endpoints.*


**Additional Features**
- [ ] Implement error handling for API requests and display meaningful
- [ ] messages to the user.
- [ ] Add basic authentication (e.g., JWT) to secure the API endpoints.
- [ ] Write unit tests
